# Remote Keyboard
A program that can be used to control another computer's keyboard remotely.

# Usage
1. Modify server_addr in [client.py line #17]()
2. Open server.py
3. Open client.py
4. Start to send keydown events

# Test Environment
+ Windows 10
+ Python 3.9.0
